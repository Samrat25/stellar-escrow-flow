## Recording Tips

1. **Use screen recording software:**
   - OBS Studio (Free)
   - Loom (Easy to use)
   - QuickTime (Mac)

2. **Prepare before recording:**
   - Have 2 test wallets ready with testnet XLM
   - Clear browser cache
   - Close unnecessary tabs
   - Test audio quality

3. **During recording:**
   - Speak clearly and explain each step
   - Show actual blockchain transactions
   - Highlight key features
   - Keep it under 10 minutes

4. **After recording:**
   - Upload to YouTube (unlisted or public)
   - Add timestamps in description
   - Update this file with the link
