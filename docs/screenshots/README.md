## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (preferred) or JPG
- **Browser**: Chrome or Firefox at 100% zoom
- **Theme**: Show both light and dark mode if applicable
- **Content**: Use real data from testnet, blur sensitive wallet addresses if needed

## How to Take Screenshots

1. Open the application in your browser
2. Navigate to the desired page
3. Use browser's full-page screenshot feature or:
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
   - Linux: Use screenshot tool

4. Save with the exact filename listed above
5. Place in this directory
